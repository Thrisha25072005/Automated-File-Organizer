# 📂 Automated File Organizer

A Python desktop application that automatically organizes files into categorized folders using CustomTkinter.

## 🚀 Features

- 📁 Organizes files automatically
- 🖼️ Separates Images
- 📄 Separates Documents
- 🎵 Separates Audio
- 🎥 Separates Videos
- 🗜️ Separates ZIP Files
- 💻 Separates Applications
- 📦 Moves unknown files to Others
- 📊 Displays organization progress
- 📝 Shows organized files in a log window
- 🎨 Modern GUI using CustomTkinter

## 🛠️ Technologies Used

- Python
- CustomTkinter
- Tkinter
- shutil
- os

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Thrisha25072005/Automated-File-Organizer.git
```

2. Install CustomTkinter

```bash
pip install customtkinter
```

3. Run the application

```bash
python organizer_gui_v4.py
```

## 📷 Output

The application organizes files into folders such as:

- 📁 Images
- 📁 Documents
- 📁 Audio
- 📁 Videos
- 📁 ZIP Files
- 📁 Applications
- 📁 Others
